IoT Gateway

Con IoT Gateway usted podrá monitorear y gestionar, con precisión y en tiempo real, toda la información relacionada a los procesos que se llevan adelante en su industria.

CARACTERISTICAS

IoT Gateway permite realizar y reportar mediciones de múltiples variables de importancia en sus procesos industriales.

IoT Gateway cuenta con 2 entradas digitales, 2 entradas analógicas, 2 salidas relé, conexión WiFi y Ethernet, soporta hasta 6 sensores de temperatura sin contacto, 24 sensores de temperatura con contacto y 2 sensores de humedad relativa.

IoT Gateway soporta comunicación Modbus TCP lo que permite la interacción con equipos como un PLC. De este modo, el equipo puede reportar el estado de todos los registros y, de forma indirecta, el funcionamiento de los equipos controlados por el PLC, por ejemplo, un variador de frecuencia, un caudalímetro, etc.

TIEMPO REAL

Todas las mediciones que IoT Gateway realiza son enviadas por medio de una conexión WiFi hacia la nube, en un intervalo de tiempo configurable. De esta manera, un supervisor de planta puede observar el estado de estas mediciones en cualquier momento y en cualquier lugar, mediante una aplicación en su teléfono móvil.

DATOS HISTORICOS

Desde la plataforma IoT usted podrá visualizar los datos históricos de cada variable medida. Allí podrá seleccionar el rango de fechas que quiere observar. Además, tendrá la posibilidad de descargar el conjunto de datos para procesarlo mediante una planilla de cálculo.

ALARMAS

Si así lo desea, puede configurar alarmas para cada una de las variables medidas. Cuando una medición se mueva por fuera del rango permitido, una notificación le llegará a su teléfono móvil indicando que hay un valor anormal. Además, algunas variables pueden ser combinadas con las salidas relé, por ejemplo, una salida puede ser activada si la temperatura supera un valor dado. En este caso el equipo funciona como un termostato.

CONFIGURACIONES

IoT Gateway es altamente configurable. Todas las configuraciones se realizan de forma local mediante un servidor web incluido en el propio equipo. Desde esta web podrá configurar, por ejemplo, si desea activar una salida relé cuando se activa una entrada digital, esto sería útil para encender una luz o sirena cuando se abre la puerta de un tablero.

ACTUALIZACIONES

IoT Gateway posee un novedoso sistema de actualización automática a través de internet. El equipo comprueba periódicamente si hay una nueva actualización disponible, de ser así, este la descarga e instala, sin que un usuario intervenga. De esta manera, el equipo siempre estará actualizado con todas las nuevas funciones que se desarrollen en el futuro.

PLATAFORMA IoT

Los usuarios podrán ver toda la información que registra y reporta el equipo, a través de una aplicación móvil, disponible para dispositivos Android y Apple. Desde esta plataforma se tendrá acceso a los valores en tiempo real, eventos y alarmas en curso, historiales de datos, entre otra información.
